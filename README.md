## Browsers Support

According to [.browserslistrc](https://github.com/kuohuanhuan/x-markdown-css/blob/master/.browserslistrc), `x-markdown-css` supports **all browsers support CSS variables.**

## Development

### Requirement

- Node.js >= 16

### Install

```sh
npm i
```

### Build

```sh
npm run build
```

or

```sh
grunt build
```

or

```sh
npx grunt build
```
